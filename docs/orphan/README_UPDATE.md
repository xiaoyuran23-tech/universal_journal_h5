## 📱 使用说明

### 记录物品

1. 点击右下角 **+** 按钮
2. 填写名称、品类、状态、备注
3. 点击右上角 **完成** 保存

### 收藏管理

1. 点击底部 **收藏** Tab
2. 浏览已收藏物品（网格布局）
3. 使用筛选下拉按品类过滤
4. 点击卡片查看详情

### 设置密码

1. 点击底部 **我** Tab
2. 选择 **隐私保护** → **密码保护**
3. 设置 4-20 位密码
4. 确认后数据将被加密存储

### 导出数据

1. 进入 **我** → **数据管理** → **导出数据**
2. 自动下载 JSON 备份文件
3. 建议定期备份
