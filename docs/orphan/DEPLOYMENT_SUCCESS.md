
### 2. 添加到主屏幕

**iOS (Safari):**
1. 打开网站
2. 点击底部「分享」按钮

## 🎯 使用流程

### 添加第一个物品

1. 点击右下角「+」悬浮按钮
2. 填写物品信息：
   - 名称（必填）
   - 主品类（植物/手办/书籍/数码/宠物/其他）
   - 子分类（可选）
   - 状态（在役/闲置/已出/绝版）
   - 备注/故事（可选）
3. 上传图片（可选）
4. 点击「保存入库」

### 搜索物品

1. 在搜索框输入关键词
2. 点击「搜索」或按回车
3. 结果实时显示

### 筛选品类

1. 点击分类筛选按钮
2. 选择要查看的品类
3. 列表自动过滤

### 切换主题

1. 点击右上角太阳图标 ☀️
2. 选择喜欢的主题
3. 即时生效，自动保存

## ⚠️ 注意事项

### 数据存储
- 数据存储在浏览器 localStorage
- 清除浏览器数据会丢失
- 建议定期导出备份（未来功能）

### 图片上传
- Base64 存储会膨胀约 33%
- 建议上传前压缩图片
- 单张图片建议 < 2MB

### 主题切换
- 主题选择存储在 localStorage
- 清除浏览器数据会重置主题
- 可手动切换回喜欢的主题
