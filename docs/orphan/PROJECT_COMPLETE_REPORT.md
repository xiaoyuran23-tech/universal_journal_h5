## 🔧 已知问题

### 1. GitHub Pages 根路径 404
**现象**: 访问 `https://.../universal_journal_h5/` 显示 404  
**原因**: GitHub CDN 缓存问题  
**解决**: 
- 使用完整路径：`https://.../universal_journal_h5/index.html`
- 或等待 10-60 分钟 CDN 自动刷新

### 2. iOS PWA 缓存
**现象**: 更新后 iOS 仍显示旧版本  
**解决**:
- 删除主屏幕图标重新添加
- 或 Safari 清除缓存

### 3. CSS 文件过大（3718 行）
**现象**: 样式文件包含 5 个主题，体积较大  
**影响**: 首次加载稍慢（~100KB）  
**优化**: 未来可考虑按需加载主题
