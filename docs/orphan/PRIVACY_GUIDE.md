# 🔐 万物手札 H5 - 隐私保护功能使用指南

## 快速开始

### 1️⃣ 设置密码保护

**步骤：**
1. 点击底部 TabBar 的「**设置**」
2. 找到「**🔒 隐私保护**」区域
3. 点击「**设置密码**」按钮
4. 输入 4-20 位密码
5. 再次输入确认密码
6. ✅ 完成！数据已加密存储

**注意事项：**
- ⚠️ **请务必记住密码**（无后端无法重置）
- 建议将密码保存在密码管理器
- 首次设置前建议先导出数据备份

---

### 2️⃣ 解锁数据

**场景：** 刷新页面或重新访问时

**步骤：**
1. 页面显示锁定遮罩 🔒
2. 输入密码
3. 点击「**解锁**」或按回车
4. ✅ 解锁成功，数据已加载

**错误处理：**
- 密码错误会显示红色提示
- 可无限次重试
- 忘记密码无法恢复数据

---

### 3️⃣ 关闭密码保护

**步骤：**
1. 进入「设置」页面
2. 点击「**关闭保护**」
3. 输入当前密码验证
4. ✅ 数据恢复明文存储

---

## 📦 数据备份与恢复

### 导出数据

**何时使用：**
- 定期备份（建议每周）
- 更换设备前
- 开启密码保护前

**操作：**
1. 进入「设置」→「数据管理」
2. 点击「**导出 JSON**」
3. 浏览器下载备份文件
4. 文件名：`万物手札备份_YYYY-MM-DD.json`

**备份内容：**
- 所有物品数据
- 名称、分类、状态
- 图片（Base64 编码）
- 时间戳

---

### 导入数据

**何时使用：**
- 恢复备份
- 多设备同步
- 合并数据

**操作：**
1. 进入「设置」→「数据管理」
2. 点击「**导入 JSON**」
3. 选择备份文件
4. 系统自动合并

**合并规则：**
- ✅ 新增：导入不存在的记录
- ⏭️ 跳过：已存在的 ID 不会覆盖
- 📊 反馈：显示新增/跳过数量

## 🔒 加密原理

### 技术实现

```
明文数据
    ↓
生成随机 Salt (16 字节)
    ↓
密钥派生 (密码 + Salt, 100 轮迭代)
    ↓
XOR 异或加密
    ↓
Base64 编码
    ↓
存储到 localStorage
```

### 安全特性

- **随机 Salt**: 每次加密都不同，防止彩虹表攻击
- **密钥派生**: 100 轮迭代增加暴力破解成本
- **XOR 加密**: 轻量级但有效的对称加密
- **Base64**: 避免特殊字符导致的存储问题

### 数据存储位置

**加密后：**
- `localStorage._uj_data_encrypted` = 加密数据 (Salt:Base64)
- `localStorage._uj_hash` = 密码哈希 (用于验证)
- `localStorage._uj_locked` = "true" (锁定状态)

**未加密：**
- `localStorage.universal_journal_items` = 明文 JSON

## 🆘 常见问题

### Q: 忘记密码怎么办？

**A:** 很遗憾，数据无法恢复。这是零知识保护的设计特性。

**预防方案：**
1. 开启保护前务必备份
2. 将密码保存在密码管理器
3. 记录在安全的物理位置

---

### Q: 导出数据后，导入会覆盖现有数据吗？

**A:** 不会。导入采用智能合并：
- 相同 ID 的记录会被跳过
- 新 ID 的记录会被添加
- 不会删除或覆盖任何现有数据

---

### Q: 加密会影响性能吗？

**A:** 几乎无感知：
- 加密/解密耗时 < 50ms
- 100 条数据加密约 30ms
- 日常使用无卡顿

---

### Q: 可以在多个设备间同步吗？

**A:** 可以，但需手动操作：

**方案 1：定期导入导出**
1. 设备 A 导出数据
2. 通过微信/QQ/邮件发送到设备 B
3. 设备 B 导入数据

**方案 2：云盘同步**
1. 将备份文件放在云盘同步文件夹
2. 各设备从云盘导入

---

### Q: 加密后数据会上传到服务器吗？

**A:** 不会。所有数据存储在本地：
- 数据在你的浏览器 `localStorage`
- 加密/解密都在本地完成
- 没有任何数据发送到服务器
