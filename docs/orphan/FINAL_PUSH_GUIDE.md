## 🎯 最佳方案：使用 GitHub Desktop

**这是最简单、最可靠的方式，不受环境变量影响**

### 步骤 1：下载 GitHub Desktop

访问：https://desktop.github.com/download/

下载安装并登录你的 GitHub 账号 (`xiaoyuran23-tech`)

### 步骤 2：克隆仓库

1. 打开 GitHub Desktop
2. File → Clone repository
3. 选择 `xiaoyuran23-tech/universal_journal_h5`
4. 选择保存路径（如 `C:\Users\你的用户名\Documents\GitHub\universal_journal_h5`）
5. 点击 Clone

### 步骤 3：复制项目文件

1. 打开文件资源管理器
2. 进入 `D:\QwenPawOut001\universal_journal_h5`
3. 选择以下文件和文件夹：
   - `index.html`
   - `css/` 文件夹
   - `js/` 文件夹
   - `assets/` 文件夹
   - `.github/` 文件夹
   - `README.md`
   - `DEPLOY_GUIDE.md`
   - `deploy.bat`
   - `deploy.sh`
   - `start.bat`
   - `start.sh`

4. 复制 (Ctrl+C)

### 步骤 4：粘贴到克隆的仓库

1. 打开 GitHub Desktop 克隆的文件夹
2. 粘贴 (Ctrl+V)
3. 如有覆盖提示，选择"是"

### 步骤 5：提交并推送

1. 回到 GitHub Desktop
2. 看到变更的文件列表
3. 输入提交信息：`📦 万物手札 H5 - 5 主题系统 + 按钮优化`
4. 点击 "Commit to main"
5. 点击 "Push origin"
