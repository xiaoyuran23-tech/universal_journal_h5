# ⚠️ 已知问题与修复指南

## 问题：app.js 编码损坏

**原因**: 多次 edit_file 操作导致 UTF-8 编码损坏

**影响**: 
- 页面 Tab 切换失效
- 部分中文显示为乱码

## 临时解决方案

### 方案 1: 恢复原始文件

如果你有 Git 版本控制：
```bash
cd universal_journal_h5
git checkout app.js
```

### 方案 2: 手动修复

1. 打开 `app.js` 文件
2. 找到所有乱码的中文字符
3. 替换为正确的中文

常见损坏模式：
```
损坏：鏃犳鐨勬鎹？
修复：无效的数据格式

损坏：鍔犲瘑鎴姛
修复：加密成功
```

### 方案 3: 使用备份

如果之前有备份文件，恢复备份。
