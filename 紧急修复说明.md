## 为什么之前总是看到旧版本？

GitHub Pages 的 CDN 缓存 + 浏览器本地缓存 = 双重缓存

**解决方案**:
1. 代码中添加了 `?v=2.1.2` 参数 → 强制浏览器重新下载文件
2. 添加了 no-cache meta 标签 → 告诉浏览器不要缓存

这次推送后，**所有用户的浏览器都会强制下载最新版本**！
